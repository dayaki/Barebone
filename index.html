Barebone

About Barebone?

Version: 1.0

Barebone is a small collection of well-organized CSS files that can help you rapidly develop sites that look beautiful at any size, be it a 17" laptop screen or an iPhone. It's based on a responsive *fluid grid, but also provides very basic CSS for typography, buttons, forms and media queries. Go ahead, resize this super basic page to see the grid in action.

Docs & Support

The easiest way to really get started with Barebone is to check out the full docs and info at www.sprypixels.com.. Barebone is also open-source and has a project on git, so check that out if you want to report bugs or create a pull request. If you have any questions, thoughts, concerns or feedback, please don't hesitate to email me at barebone@sprypixels.com.

Three Core Principles

Barebone is built on three core principles:

  • A Responsive Fuild Grid Down To Mobile: Uses Fuild(Percentage based) rather than adaptive(pixels based) grid. Elegant scaling from a browser to tablets to mobile.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten.
